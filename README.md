# Graph It!

A simple graph visualization tool made for easy and direct graphing in the browser.

It is made for quick math plotting, live updates, and a clean look.

## What it does

- Live graphing while you type
- Multiple expressions
- Global parameters
- Time-based graphs with `t`
- Dark UI

## How to use

Open `index.html` in your browser and start typing math expressions.

You do not need to press any extra button to update the graph.

## Basic examples

- `sin(x)`
- `cos(x)`
- `sin(x * t)`
- `a * sin(b * x)`

## Parameters

You can add parameters like:

- `a = 5`
- `b = 10`

Then use them in your expressions.

## Controls

- `Ctrl + Z` to undo
- `Ctrl + Y` to redo

## Notes

- The app is meant to stay light and fast
- Empty or unfinished expressions are ignored
- All the main settings are in `app.js`

## Tweakable stuff

If you want to change how it looks or behaves, check the top of `app.js`.
